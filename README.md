# Fashion Store

Fashion Store is a full-stack e-commerce web application developed using Java, JSP, Servlets, JDBC, MySQL, HTML, CSS, and JavaScript.

## Features

- User Registration & Login
- Product Listing
- Product Details
- Add To Cart
- Update Cart Quantity
- Checkout
- Place Order
- My Orders
- Order Details
- Stock Management

## Technologies Used

- Java
- JSP
- Servlets
- JDBC
- MySQL
- HTML
- CSS
- JavaScript
- Apache Tomcat 10.1

## Database Configuration

Update DBConnection.java with your MySQL username and password before running the project.

Database Name: fashionstore
